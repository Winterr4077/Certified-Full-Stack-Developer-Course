CS2 C4

The one and only detailed guide to making a not so safe, c4 bomb from CS2.

Ingredients

  • Explosive blocks
  • Gorrila tape
  • Circuit board
  • Nokia keypad (4x4)
  • Old calculator display
  • Wires
  • Battery pack

Instructions

  1. Lay out your explosive blocks
  2. Wrap everything in Gorilla tape
  3. Slap on the circuit board
  4. Throw your old calculator screen on the top right of the explosive blocks
  5. Add wires, it doesn't matter if they're connected, it'll still work, trust me.
  6. Attach the battery pack
  7. Punch the code 7355608 and pray solider.
